# 1. 최대 공약수

유클리드 호제법

1~Math..sqrt(x) 까지 나눠서 떨어지는 거면 약수임

n의 모든 약수들을 구하기 -> 내림차순 정렬 -> m % 약수 == 0 인거 고르기  



```java
public static int euclid(int a, int b) {
    while(b != 0) {
        int r = a % b;
        a = b;
        b = r;
    }
    
    return a;
}
```
