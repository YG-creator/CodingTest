# String[] -> int[] 변환

```java
import java.util.Arrays;

String[] = {"1","2","3"};
int[] arr =  Arrays.stream(a).map(String::trim).mapToInt(Integer::parseInt).toArray();
```



# 반올림


# 올림


# 내림

