# 스택



## 개념

LIFO인 선형 자료구조



## java 코드

```java
import java.util.stack;

Stack<자료형> stack = new Stack<>();

stack.add();
stack.pop();	// 비어있을 때 고려
stack.peek();
```

# 큐



## 개념

FIFO인 선형 자료구조



## java 코드

```java
import java.util.Queue;

Queue<Integer> q = new LinkedList<>();

q.add();
q.poll();	// 비어있을 때 고려
q.peek();
```



## 문제 유형

FIFO - [카드 2 #2164](https://www.acmicpc.net/problem/2164), 2164 카드2, 

원순열 k번째 값 - 1158 요세푸스 문제
