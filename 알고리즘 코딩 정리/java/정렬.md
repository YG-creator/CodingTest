# 2차 행렬 정렬

```java
//2차 배열 정렬(3번째열 기준, 내림차순)
Arrays.sort(graph, new Comparator<int[]>(){
    @Override
    public int compare(int[] o1, int[] o2) {
        return o2[2]-o1[2];
    }
});
```
