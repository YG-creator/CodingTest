# 우선순위 큐 정렬

```java
class Edge implements Comparable<Edge>{	// 우선순위 큐 정렬설정
	// 변수
	int s;	
	int e;
	double c;
	
	public Edge(int s, int e, double c) {	// 생성자
		this.s = s;
		this.e = e;
		this.c = c;
	}

	@Override
	public int compareTo(Edge o) {	// 비교 오름차순(double형)
		// TODO Auto-generated method stub
		return Double.compare(this.c, o.c);
	}	
}
```
