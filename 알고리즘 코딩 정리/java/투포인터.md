# 개념

구간합 구하기 or 두수의 합 or 연속된 구간 합 or 간격일정 연속된 구간 합

1. 다 더한후 두지점 빼기 -> SUM(1,N) - SUM (1,M) = SUM(M+1,N)

2. 정렬 후 -> 값이 크면 e = e-1, 값이 작으면 s = s+1

3. s,e= 1부터 시작 -> 값이 작으면 e++  /  값이 크면 s++ -> s,e != n 까지 반복
4. 행렬 연장(n-1만틈) -> e = s+n  -> 하나씩 증가
