# 트리의 순회

## 전위(PreOrder)

루트->왼->오



## 중위(InOrder)

왼->루트->오



## 후위(PostOrder)

왼->오->루트



# 문제

* 중위+후위 -> 전위

  2263 트리의 순회

​		postOrder 트리의 마지막 값이 트리의 루트

​		inOrder 트리에서 부모노드의 왼쪽이 자식노드, 오른쪽이 오른쪽노드임
