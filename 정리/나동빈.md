## 냅색

무게제한, 가치 최대

1. dp 만들기 `new int[n+1][k+1]`

2. 값 arr 입력  `w[], v[]`

3. 무게가 k무게 이상이면 최댓값 구하기 `dp[i][j] = max(dp[i-1][j],dp[i-1][j-k] + v)`

4. k무게 이하면 위에거 넣기`dp[i][j] = dp[i-1][j]`
